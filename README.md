# BelajarYuk

BelajarYuk adalah aplikasi pembelajaran adaptif berbasis AI yang menyesuaikan materi dengan gaya belajar siswa. Proyek ini dibuat oleh tim dari SMK STRADA Jakarta sebagai solusi atas tantangan pendidikan di era digital.

## 🚀 Fitur Utama
- Tes gaya belajar otomatis
- Rekomendasi materi adaptif berbasis AI
- Sistem kuis dinamis

---

## 🛠️ Teknologi yang Digunakan
- Frontend: React.js
- Backend: HapiJS
- Machine Learning: Python
- Database: MongoDB
- Desain UI: Figma
- Deployment: Localhost / VPS

---

## 📦 Cara Instalasi (Lokal)

### 1. Clone Repository
```bash
git clone https://github.com/HusniAja211/BelajarYuk.git

cd belajaryuk
```

### 2. Jalankan Frontend (React)
Buka terminal baru:
```bash
cd frontend
npm install
npm run dev
```

### 4. Akses Aplikasi
Frontend: Gunakan tautan yang diberikan oleh vite.  


---

## 📘 Panduan Deploy Produksi
Untuk menjalankan versi produksi, Anda bisa menggunakan VPS seperti Render atau Railway, atau deploy ke Heroku untuk backend dan Netlify/Vercel untuk frontend.
